# Adidas Sales Analysis Dashboard (Power BI)

## Project Overview
This project presents an interactive Adidas Sales Analysis Dashboard developed using Power BI. The dashboard analyzes sales performance across different products, regions, retailers, and time periods to uncover key business insights.

The purpose of this project is to demonstrate how business intelligence tools can be used to analyze sales data, monitor performance metrics, and support data-driven decision making.

The dashboard provides interactive visualizations that allow users to explore sales trends, identify top-performing products, evaluate regional performance, and analyze retailer profitability.

## Problem Statement
Retail companies generate large volumes of sales data from different regions, products, and retailers. Without proper analysis, it becomes difficult to identify profitable products, understand regional performance, and track sales trends.

This project analyzes Adidas sales data to answer key business questions such as:

- Which products generate the highest sales and profit?
- Which retailers contribute the most revenue?
- How do sales change over time?
- Which regions perform better in terms of sales and profit?

## Dataset Description
The dataset contains Adidas retail sales data including the following attributes:

- Retailer
- Product Category
- Sales Method
- Region
- State
- City
- Units Sold
- Total Sales
- Operating Profit
- Price per Unit
- Date

The dataset represents sales transactions across multiple locations and retailers.

## Tools and Technologies
- Power BI
- Power Query
- Data Modeling
- DAX (Data Analysis Expressions)
- Data Visualization

## Data Preparation
Data preparation was performed using Power Query with the following steps:

- Removed duplicate records
- Handled missing values
- Converted columns to correct data types
- Created calculated columns and measures
- Structured the dataset for better performance in Power BI

## Key Performance Indicators (KPIs)

The dashboard highlights the following business metrics:

- Total Sales: 120 Million
- Units Sold: 2 Million
- Average Price per Unit: 45.22
- Total Profit: 47.22 Million

These KPIs provide a quick overview of overall sales performance.

## Dashboard Pages

### Home Dashboard
Provides a summary of overall sales performance including:

- Total Sales
- Units Sold
- Average Price
- Total Profit
- Sales by Sales Method
- Profit by Retailer

### Time Analysis
Analyzes sales performance across time including:

- Monthly sales filtering
- Units sold trend over time
- Sales trend analysis

This page helps identify seasonal sales patterns and demand fluctuations.

### Product Analysis
Provides insights about product categories including:

- Units sold by product and gender
- Profit by product category
- Sales distribution by product type

This helps identify which products perform best in the market.

### Regional Analysis
Focuses on geographic performance including:

- Total sales by city (map visualization)
- Sales by region
- Profit by city
- Regional sales comparison

This helps identify high-performing regions and locations.

### Product Details
Provides detailed transactional insights including:

- Sales trend over time
- Retailer-wise sales
- Detailed sales table with location and retailer information

This allows deeper exploration of sales data.

## Key Insights

- Street Footwear generates the highest sales and profit among product categories.
- Certain retailers contribute significantly more revenue than others.
- Sales performance varies significantly across regions.
- Some cities consistently generate higher profits.
- Sales trends fluctuate across different months, indicating seasonal demand.
